1、左侧菜单：

• 点击“API文档”：加载全部应用下的全部功能下的API全集。

• 点击某API名称：进入ＡＰＩ详情页。


2、ＡＰＩ列表：

•  列表标题：

   “API文档”为入口，标题为“全部”。

    “应用”为入口，标题为应用名称。

    “功能”为入口，标题为功能名称。

•  展示字段：API名称、描述（ＡＰＩ简介）

• 排序：创建时间倒序

• API名称：点击，进入API详情页



注：用户未登录，允许查看API列表，点击“申请”按钮为检查身份及其他条件时机。
